Sobre o Fundação

Nossa missão é promover educação de qualidade acerca da linguagem de programação C, mediante a utilização de materiais teóricos, exercícios e juiz eletrônico integrado com inteligência artificial para envio de atividades.

O projeto FUNDAÇÃO é composto por três módulos. Sendo eles:

MÓDULO I

  • introdução à algoritmos
  • operadores
  • condicional
  • repetição

MÓDULO II

  • vetores
  • strings
  • manipulação de strings
  • matrizes

MÓDULO III

  • funções
  • struct
  • ponteiros
  • depuração

Tal divisão foi realizada para aprimorar o aprendizado gradativo, introduzindo a lógica de programação e as ferramentas necessárias para implementação.
